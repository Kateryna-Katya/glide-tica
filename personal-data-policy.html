Политика касаемо обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки персональных данных, предпринимаемые платформой GlideTica (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных.

1.1. Важнейшей целью и условием осуществления своей деятельности Оператор считает соблюдение прав и свобод человека и гражданина при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни в цифровой среде ЕС.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта glide-tica.blog.

2. Основные понятия

  • Веб-сайт — совокупность графических и информационных материалов, доступных в сети интернет по адресу glide-tica.blog.
  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся прямо или косвенно к Пользователю сайта.
  • Обработка персональных данных — сбор, запись, накопление, хранение и удаление данных с использованием передовой цифровой инфраструктуры.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Данные, предоставляемые Пользователем:
    • Имя и фамилия;
    • Адрес электронной почты (Email);
    • Номера телефонов.
  • Данные, собираемые автоматически:
    • Обезличенные данные о посетителях (в т.ч. файлы «cookie») с помощью сервисов интернет-статистики для улучшения качества работы платформы.

4. Цели обработки

  • Идентификация Пользователя для предоставления инновационных карьерных сервисов.
  • Установление обратной связи для проведения бесплатных консультаций.
  • Информирование Пользователя посредством электронных рассылок о новых стратегиях роста.

5. Правовые основания

Оператор обрабатывает данные только при наличии добровольного согласия Пользователя, выраженного путем заполнения форм на сайте glide-tica.blog.

6. Безопасность данных

  • Оператор принимает все возможные технические меры для защиты данных в соответствии с регламентами GDPR.
  • Персональные данные никогда не будут переданы третьим лицам без согласия Пользователя, за исключением случаев, предусмотренных законом.
  • Пользователь может в любой момент отозвать согласие, направив письмо на hello@glide-tica.blog.

7. Заключительные положения

Актуальная версия Политики находится в свободном доступе на сайте glide-tica.blog. Любые разъяснения можно получить по электронной почте: hello@glide-tica.blog.
